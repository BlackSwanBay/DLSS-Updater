Version 1.0.0
- Initial release with Steam support.

Version 1.1.0
- Added support for EA Launcher.
- Added support for Ubisoft Launcher.
- Added support for the Epic Games Launcher.
- Improved logging for updated and skipped games.
- Integration of a one click executable for running the application.

Version 1.2.0
- Support for Battle.Net.
- Support for GOG Launcher.
- Better summarisation of games at the end.

Version 1.2.1
- Fixes for registry path locations for game directories.

Version 1.2.2
- Change of approach for the Epic Games Launcher and EA Play to successfully detect them.

Version 1.2.4
- Revisions to logging to improve the execution flow and stop the bombarding of information to the user.
- Changes to how some launchers are detected for games.

Version 1.2.5

Version 1.2.6
